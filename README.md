# Wheater-App
an iOS app build in swift used openwheatermap API

## Feature
- Get current wheater for Jakarta only

## Library Used

- Alamofire 5.2
- RxSwift 6.2.0
- RxCocoa 6.2.0

## Tech Stack

- Clean Architecture
- UIKit
- MVVM
- RX Swift
- Swift 5
- Fetching data from API
- Persistence data using Userdefault
